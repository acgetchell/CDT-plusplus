Detailed Description

Example bistellar fip.

Author
Adam Getchell

Show how to use the bistellar_flip functions on a 3D triangulation. Some convenience functions are defined here because the internal functions of the Triangulation_3 class are not currently accessible to the bistellar_flip functions.
