30 R"(Causal Dynamical Triangulations in C++ using CGAL.
31
32Copyright (c) 2013-2026 Adam Getchell
33
34A program that generates d-dimensional triangulated spacetimes
35with a defined causal structure and evolves them according
36to the Metropolis algorithm. Specify the number of passes to control
37how much evolution is desired. Each pass attempts a number of ergodic
38moves equal to the number of simplices in the simulation.
39
40Usage:./cdt (--spherical | --toroidal) -n SIMPLICES -t TIMESLICES
41 [-d DIM]
42 [--init INITIAL RADIUS]
43 [--foliate FOLIATION SPACING]
44 [--no-output]
45 [--seed SEED]
46 -k K
47 --alpha ALPHA
48 --lambda LAMBDA
49 [-p PASSES]
50 [-c CHECKPOINT]
51
52Optional arguments are in square brackets.
53
54Examples:
55./cdt --spherical -n 32000 -t 11 --alpha 0.6 -k 1.1 --lambda 0.1 --passes 1000
56./cdt -s -n32000 -t11 -a.6 -k1.1 -l.1 -p1000 --seed 92
57
58Options)"};
